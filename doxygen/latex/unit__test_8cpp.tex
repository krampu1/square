Also, this function outputs information about the passed tests and the result of the check to the console.~\newline
~\newline
 Данная функция вызывается при указании флага компиляции -\/D TEST .~\newline
Она содержит набор тестов и проверяет на них работу основной программы с помощью функции \mbox{\hyperlink{quadratic__equation_8h_a13a9924ed63c3d94a2337babb2a6b475}{testqe()}}.~\newline
