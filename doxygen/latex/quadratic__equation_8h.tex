This function divides the quadratic equation into 2 cases.~\newline
If a is 0, then the equation is treated as linear.~\newline
If a is not equal to 0, then the equation is considered as a square equation that cannot have an infinite number of roots.~\newline
~\newline
 Эта функция разделяет квадратное уравнение на 2 случая.~\newline
Если a равен 0, то уравнение рассматривается как линейное.~\newline
