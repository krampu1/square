Function called for code testing. 

This function is called when you specify the start of the program -\/d .~\newline
It contains a set of tests and checks the operation of the main program on them using the \mbox{\hyperlink{quadratic__equation_8h_a9b7fd1c9c6785668832b407db30181b0}{testqe()}} function.~\newline
Also, this function outputs information about the passed tests and the result of the check to the console.~\newline
~\newline
 Данная функция вызывается при указании запуска программы -\/d .~\newline
Она содержит набор тестов и проверяет на них работу основной программы с помощью функции \mbox{\hyperlink{quadratic__equation_8h_a9b7fd1c9c6785668832b407db30181b0}{testqe()}}.~\newline
Так же эта функция выводит в консоль информацию о пройденных тестах и результат проверки.
