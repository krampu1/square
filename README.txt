Программа умеет решать квадратное уравнение и производить автомотическое тестирование кода.

-------------------------ДЛЯ ПОЛЬЗОВАТЕЛЯ--------------------------------------------------------

Для запуска пользовательского интерфейса нужно запустить square.exe

-------------------------ДЛЯ РЕДАКТОРА КОДА------------------------------------------------------

Для компиляции изменнёного кода следует использовать утилиту make.
Если требуется использовать встроенное тестирование нужно при запуска программы указать флаг -d

-------------------------Установка make----------------------------------------------------------

Для установки make на linux нужно использовать стандартную систему установки
пакетов и скачать непосредственно make.

Для установки make на windows нужно открыть консоль от имени администратора и ввести команду
@powershell -NoProfile -ExecutionPolicy unrestricted -Command "iex ((new-object net.webclient).DownloadString('https://chocolatey.org/install.ps1'))" && SET PATH=%PATH%;%systemdrive%\chocolatey\bin
после завершения перезапустить консоль и ввести команду
choco install make
